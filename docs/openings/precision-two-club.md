# Precision 2♣ Opening

### Opening Description

- **Strength**: 11-14/15 HCP
- **Suit Distribution**: 6+ clubs, or 5+ clubs with a four-card major

??? note "Response Structure"
    ### Response Structure

    | Response  | Meaning                                        |
    | --------- | ---------------------------------------------- |
    | 2♦        | Relay, asking for shape; at least invitational |
    | 2♥/2♠     | 5+, non-forcing                                |
    | 2NT       | Blocking with fit, or 5-5 excluding clubs      |
    | 3♣        | Natural, game invitation                       |
    | 3♦/3♥/3♠  | 6+, game invitation                            |
    | 3NT       | Natural                                        |
    | 4♣        | Blocking                                       |
    | 4♦        | Kickback; asking for aces with clubs agreed    |
    | 4♥/4♠     | To play                                        |

??? note "2♣ – 2♦ (Relay)"
    ### 2♣ – 2♦ (Relay)

    The 2♦ relay asks opener to describe shape. It does not promise a four-card major.

    | Opener's Rebid | Meaning                                                              |
    | -------------- | -------------------------------------------------------------------- |
    | 2♥             | Four hearts (maximum or minimum)                                     |
    | 2♠             | Four spades (maximum or minimum)                                     |
    | 2NT            | Six clubs, no singleton; maximum                                     |
    | 3♣             | Six clubs; minimum                                                   |
    | 3♦/3♥/3♠       | Maximum; singleton in the bid suit; model shape 6-3-3-1              |
    | 3NT            | Solid club suit, no singleton; maximum                               |

    ??? note "After 2♣ – 2♦ – 2♥ (Four Hearts)"
        #### After 2♣ – 2♦ – 2♥

        | Responder's Bid | Meaning                                                                        |
        | --------------- | ------------------------------------------------------------------------------ |
        | 2♠              | Natural (5 spades), forcing; or third suit showing honor concentration without a diamond stop |
        | 2NT             | Further inquiry                                                                |
        | 3♣              | 9-11 with club fit (no four-card major required) |
        | 3♦              | Natural, forcing                                 |
        | 3♥              | Game invitation                                  |

    ??? note "After 2♣ – 2♦ – 2♠ (Four Spades)"
        #### After 2♣ – 2♦ – 2♠

        | Responder's Bid | Meaning                                          |
        | --------------- | ------------------------------------------------ |
        | 2NT             | Further inquiry                                  |
        | 3♣              | 9-11 with club fit (no four-card major required) |
        | 3♦/3♥           | Natural, forcing                                 |
        | 3♠              | Game invitation                                  |

    ??? note "After 2♣ – 2♦ – 2NT (Six Clubs, No Singleton, Maximum)"
        #### After 2♣ – 2♦ – 2NT

        | Responder's Bid | Meaning                                                                                          |
        | --------------- | ------------------------------------------------------------------------------------------------ |
        | 3♣              | Non-forcing                                                                                      |
        | 3♦              | Natural, or attempt to check major stops; opener bids 3NT with stops in both majors, otherwise 3♥ or 3♠ |
        | 3♥/3♠           | Natural, trying to agree a suit                                                                  |
        | 4♣              | Forcing, cue-bid invitation                                                                      |
        | 4♦              | 6+, natural, with interest; cue-bid invitation (2NT denies singleton, confirming suit agreement) |

        #### Notes
        - 3♣ is non-forcing because East sometimes bids 2♦ relay with a weak hand looking for a four-card major.
        - A new suit and 2NT are game-forcing.

    ??? note "After 2♣ – 2♦ – 3♣ (Six Clubs, Minimum)"
        #### After 2♣ – 2♦ – 3♣

        | Responder's Bid | Meaning                                                                       |
        | --------------- | ----------------------------------------------------------------------------- |
        | 3♦              | Natural, or attempt to check major stops                                      |
        | 3♥/3♠           | Natural, trying to agree a suit                                               |
        | 4♣              | Forcing, cue-bid invitation                                                   |

    ??? note "After 2♣ – 2♦ – 3♦/3♥/3♠ (Maximum, Singleton in Bid Suit)"
        #### After 2♣ – 2♦ – 3♦/3♥/3♠

        Opener shows maximum strength with a singleton in the bid suit; model shape 6-3-3-1.

        | Responder's Bid | Meaning                                            |
        | --------------- | -------------------------------------------------- |
        | 3♥/3♠           | Natural, with interest; without interest sign off in 4♥/4♠ |
        | 4♣              | Forcing, cue-bid invitation                        |
        | 4♦              | Kickback; asking for aces with clubs agreed        |

??? note "2♣ – 2NT (Blocking or 5-5 Excluding Clubs)"
    ### 2♣ – 2NT

    The 2NT response is ambiguous:
    1. Blocking with a club fit.
    2. 5-5 in two suits excluding clubs (game-forcing).

    Opener must rebid 3♣. Responder then clarifies the two-suited hand:

    | Responder's Bid | Meaning              |
    | --------------- | -------------------- |
    | 3♦              | Diamonds and hearts  |
    | 3♥              | Hearts and spades    |
    | 3♠              | Spades and diamonds  |

    #### Notes
    - Pass by responder after 3♣ confirms the blocking variant.
    - A new suit and 2NT after opener's rebid are game-forcing.

??? note "Bidding After Intervention"
    ### Bidding After Intervention

    #### After a Take-Out Double
    Bid as though there was no double. Redouble is for penalties.

    #### After a Suit Overcall
    - Negative doubles apply up to 4♦.
    - 2NT is natural.
    - A new suit at the three level is forcing.
